Interactive Molecular Modelling and Visualisation Tutorial

Step 1: Introduction

This tutorial focuses on structural visualisation using SwissPDB Viewer (also known as DeepView), allowing you to explore and analyze molecular structures you've designed or obtained.

Step 2: Download SwissPDB Viewer

Download the SwissPDB viewer from the official website.

Download SwissPDB

Step 3: Install and Launch SwissPDB Viewer

Unzip the downloaded file to your Desktop and double-click the SwissPDB icon to launch.

You should see a splash screen appear, followed by the main program interface.

Step 4: Swiss-PdbViewer Interface Overview

When you launch Swiss-PdbViewer, you'll see the main interface with a toolbar at the top. This toolbar contains various tools for manipulating and analyzing molecular structures.

The toolbar is divided into several sections:

  • File operations and general controls (left side)
  • Measurement and analysis tools (center)
  • Manipulation tools like rotate, translate, and zoom (right side)

We'll focus on some key tools in the following steps.

Step 5: Open Your Custom Structure

Now, let's open a molecular structure that you've designed or obtained:

  1. Ensure you have a structure file (.pdb format) from ProBuilder or AlphaFold saved on your computer.
  2. In Swiss-PdbViewer, click on "File" in the top menu.
  3. Select "Open" from the dropdown menu.
  4. Navigate to the location where you saved your structure file.
  5. Select the file and click "Open".

Your structure should now be visible in the main viewing window.

Step 6: Basic Manipulation Tools

Now that your structure is loaded, let's explore the basic manipulation tools:

Movement Mouse Control Description Toolbar Button
Rotate Left mouse button Up/Down rotates about x-axis. Left/Right rotates about y-axis.
Translate Right mouse button Drag the molecule in any direction.
Zoom Both mouse buttons Move up to zoom out, down to zoom in.

Step 7: Centre View

Use the leftmost toolbar button to center your molecule in the viewing window.

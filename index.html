Interactive Molecular Modelling and Visualisation Tutorial

Step 1: Introduction

Welcome to this comprehensive tutorial on molecular modelling and visualisation using SwissPDB Viewer (also known as DeepView). Understanding the 3D structure of proteins is crucial for comprehending their function. While DNA and protein sequence information provide valuable insights, exploring the 3D structure allows us to explain observations that other analyses cannot account for.

In this tutorial, you'll learn how to:

  • Download and install SwissPDB Viewer
  • Navigate the user interface
  • Open and manipulate molecular structures
  • Use different visualization techniques
  • Work with the Control Panel
  • Apply various coloring schemes

This knowledge is fundamental in fields like structural biology and drug discovery, where computational biology plays a crucial role. By the end of this tutorial, you'll be able to visualize and analyze protein structures with confidence.

Step 2: Download SwissPDB Viewer

SwissPDB Viewer is a powerful, free software package developed by the Swiss Institute for Bioinformatics. It offers a wide range of visualization options and even allows for homology modelling and energy minimization. Let's start by downloading the software:

  1. Click on the "Download SwissPDB" button below. This will take you to the official download page.
  2. On the download page, you may need to accept the terms and conditions. Read through them carefully.
  3. Choose the appropriate version for your operating system (Windows, Mac, or Linux). Make sure you select the version that matches your system architecture (32-bit or 64-bit).
  4. Once you click the download link, the file should start downloading automatically. Depending on your internet speed, this may take a few minutes.
  5. After the download is complete, locate the file in your Downloads folder. It will typically be a compressed file (e.g., .zip for Windows, .tar.gz for Mac/Linux).

Step 3: Install and Launch SwissPDB Viewer

Now that you've downloaded SwissPDB Viewer, let's install and launch it. The process may vary slightly depending on your operating system:

For Windows:

  1. Locate the downloaded zip file in your Downloads folder.
  2. Right-click on the zip file and select "Extract All..."
  3. Choose a destination folder for the extracted files (e.g., your Desktop or Documents folder).
  4. Open the extracted folder and look for an executable file named "spdbv.exe".
  5. Double-click on "spdbv.exe" to launch the program.

Action: Click anywhere on the splash screen to make it disappear and access the main program interface. If you don't click, the splash screen will automatically close after a few seconds.

Step 4: Swiss-PdbViewer Interface Overview

After dismissing the splash screen, you'll see the main interface of Swiss-PdbViewer. Let's familiarize ourselves with the key components:

The interface consists of several important areas:

  1. Menu Bar: At the very top, you'll find menus like File, Edit, Select, Build, etc. These contain various commands and options for working with your molecular structures.
  2. Toolbar: Below the menu bar, there's a toolbar with icons for quick access to common functions. The toolbar is divided into several sections:
    • File operations and general controls (left side): These include options to open files, save your work, and perform general operations.
    • Measurement and analysis tools (center): Tools for measuring distances, angles, and performing various analyses on your molecules.
    • Manipulation tools (right side): These include options to rotate, translate, and zoom your view of the molecule.
  3. Main Viewing Window: The largest part of the interface where your molecular structures will be displayed in 3D.
  4. Control Panel: Usually located to the side or as a separate window, this panel allows you to manipulate individual residues or groups of residues in your structure.
  5. Status Bar: At the bottom of the window, providing information about the current state of the program or selected elements.

Take a moment to look around the interface and familiarize yourself with these components. In the following steps, we'll dive deeper into using these tools to visualize and analyze molecular structures.

Step 5: Open Your Custom Structure

Now that we're familiar with the interface, let's open a molecular structure. In this tutorial, we'll assume you have a structure file that you've designed using ProBuilder or obtained from AlphaFold. If you don't have such a file, you can download a sample PDB file from the Protein Data Bank (www.rcsb.org) for practice.

To open your structure:

  1. Ensure you have a structure file (.pdb format) saved on your computer. PDB (Protein Data Bank) format is a standard file format for representing 3D structures of proteins and nucleic acids.
  2. In Swiss-PdbViewer, click on "File" in the top menu.
  3. Select "Open" from the dropdown menu. Alternatively, you can use the keyboard shortcut Ctrl+O (Windows) or Cmd+O (Mac).
  4. In the file browser that appears, navigate to the location where you saved your structure file.
  5. Select the file and click "Open". You may see a loading progress bar as the program reads the file.

After a moment, your structure should appear in the main viewing window. Here are a few things to note:

  • If the structure doesn't automatically center in the view, don't worry – we'll cover how to adjust the view in the next step.
  • Depending on the size and complexity of your structure, it may take a few seconds to load, especially on older computers.
  • By default, the structure is usually displayed in a line or wireframe representation. We'll learn how to change this in later steps.
  • If you open multiple PDB files, they will be loaded into separate layers, which can be managed through the Control Panel.

Troubleshooting: If you encounter an error when opening your file, ensure it's a valid PDB format. Some common issues include:

  • File is corrupted or incomplete
  • File is in a different molecular file format (e.g., .mol2, .xyz)
  • File contains unsupported elements or non-standard residues
If problems persist, try opening a known good PDB file from the Protein Data Bank to verify that SwissPDB Viewer is functioning correctly.

Step 6: Basic Manipulation Tools

Now that your structure is loaded, let's explore the basic tools for manipulating your view of the molecule. These tools are essential for examining different parts of your structure and understanding its 3D arrangement.

Movement Mouse Control Description Toolbar Button
Rotate Left mouse button Click and drag to rotate the structure. Moving up/down rotates around the x-axis, while moving left/right rotates around the y-axis. This allows you to view the molecule from different angles.
Translate Right mouse button Click and drag to move the structure within the viewing window. This is useful for centering specific parts of your molecule or moving it to a better viewing position.
Zoom Both mouse buttons Click and drag both mouse buttons simultaneously. Moving up zooms out, moving down zooms in. This allows you to examine details or view the overall structure. Alternatively, you can use the scroll wheel on your mouse.

Practice exercise: Try these movements with your loaded structure:

  1. Rotate the molecule 360 degrees around the vertical axis.
  2. Zoom in to examine a specific region, then zoom out to see the whole structure.
  3. Use the translate function to move the molecule to different areas of the viewing window.

Remember, you can always reset your view if you get lost – we'll cover that in the next step.

Step 7: Centre View

After manipulating your view, you might want to reset it to a standard position. The "Centre View" function is perfect for this:

This button, typically the leftmost on the toolbar, performs several actions simultaneously:

  • It adjusts the translation of your molecule so that it's centered in the viewing window.
  • It adjusts the zoom level so that the entire molecule is visible.
  • It resets the center of rotation to the middle of the visible structure.

Action: Try using the Centre View button now. Notice how it repositions your molecule for optimal viewing.

Tip: Use this function whenever you feel "lost" in your structure or when you want to start examining it from a standard view.

Step 8: The Control Panel

The Control Panel is a powerful tool for manipulating your structure at a more detailed level. It allows you to control the display of individual residues or groups of residues.

If you can't see the Control Panel:

  1. Go to the "Window" menu in the top toolbar.
  2. Select "Control Panel" from the dropdown list.

The Control Panel provides detailed information about your structure and allows you to change various display options. Here's what each column in the Control Panel means:

  • Group: This column describes each residue in your structure. The format is typically [Chain][Secondary Structure][Amino Acid][Position]. For example, "A s GLY95" means Chain A, part of a β-sheet (s), Glycine at position 95.
  • Show: Toggles visibility of backbone atoms for each residue. A checkmark means the backbone is visible.
  • Side: Toggles visibility of side chains for each residue. A checkmark means the side chain is visible.
  • Labl: Adds a floating label next to the residue in the 3D view. Useful for identifying specific residues.
  • :: Shows whether each residue has an associated surface. This is used when displaying molecular surfaces.
  • Ribn: Toggles ribbon representation for each residue. Useful for visualizing secondary structure.
  • Col: Indicates the color applied to each residue. You can change colors here.

You can click on these columns to toggle options on or off for individual residues or entire sections of your structure.

Practice: Try toggling different options in the Control Panel and observe how it affects the display of your molecule in the main viewing window.

Step 9: Making Selections with the Control Panel

The Control Panel is not just for viewing information; it's also a powerful tool for selecting specific parts of your structure. This is crucial for many operations, such as changing colors or representations of specific regions.

Here's how you can make selections:

  • Select a single residue: Simply click on the residue name in the "Group" column.
  • Select all residues in a chain: Click on the chain identifier at the start of the chain in the "Group" column.
  • Select all residues in a secondary structure element: Click on the secondary structure identifier (H for helix, S for sheet, etc.) in the "Group" column.
  • Select multiple residues: Click and drag over multiple rows in the Control Panel.
  • Add to a selection: Hold Ctrl (Windows) or Cmd (Mac) while clicking to add to your current selection.
  • Select a range: Click the first residue, then hold Shift and click the last residue to select all residues in between.

Selected residues are highlighted in red in the Control Panel. In the 3D view, selected residues are typically highlighted or displayed differently, depending on your current visualization settings.

Practice exercise: Try the following selections:

  1. Select all residues in an alpha helix.
  2. Select a range of 10 consecutive residues.
  3. Select all glycine residues in your structure.

Tip: You can also use the "Select" menu in the top toolbar for more advanced selection options, such as selecting by residue type or solvent accessibility.

Step 10: Different Structural Representations

SwissPDB Viewer offers various ways to represent molecular structures, each highlighting different aspects of the molecule. Understanding these representations is crucial for effective structural analysis.

To change the representation of your molecule:

  1. Select the part of the molecule you want to change using the Control Panel (as learned in Step 9).
  2. Go to the "Display" menu in the top toolbar.
  3. Choose the desired representation from the options available.

Practice exercise: Try displaying different parts of your molecule in different representations. For example:

  • Show the backbone in ribbon representation
  • Display a specific residue in ball and stick representation
  • Use spacefilling representation for surface residues

Tip: Combining different representations can be very powerful for highlighting specific features of your molecule while maintaining context of the overall structure.

Interactive Molecular Modelling and Visualisation Tutorial

Step 1: Introduction

This tutorial focuses on structural visualisation using SwissPDB Viewer (also known as DeepView), allowing you to explore and analyze molecular structures you've designed or obtained.

Step 4: Swiss-PdbViewer Interface Overview

When you launch Swiss-PdbViewer, you'll see the main interface with a toolbar at the top. This toolbar contains various tools for manipulating and analyzing molecular structures.

The toolbar is divided into several sections:

  • File operations and general controls (left side)
  • Measurement and analysis tools (center)
  • Manipulation tools like rotate, translate, and zoom (right side)

We'll focus on some key tools in the following steps.

Step 5: Open Your Custom Structure

Now, let's open a molecular structure that you've designed or obtained:

  1. Ensure you have a structure file (.pdb format) from ProBuilder or AlphaFold saved on your computer.
  2. In Swiss-PdbViewer, click on "File" in the top menu.
  3. Select "Open" from the dropdown menu.
  4. Navigate to the location where you saved your structure file.
  5. Select the file and click "Open".

Your structure should now be visible in the main viewing window.

Step 6: Basic Manipulation Tools

Now that your structure is loaded, let's explore the basic manipulation tools:

Step 9: The Control Panel

The Control Panel is a crucial tool for manipulating your structure. If you can't see this window, select Window â†’ Control Panel from the Window menu on the main toolbar.

The Control Panel provides information about your structure and allows you to change various display options for some or all of the residues. Here's what each column means:

  • Group: Describes a residue in your structure (chain, secondary structure, amino acid, position).
  • Show: Toggles visibility of backbone atoms for each residue.
  • Side: Toggles visibility of side chains.
  • Labl: Adds a floating label next to the residue.
  • :: Shows whether each residue has an associated surface.
  • Ribn: Toggles ribbon representation for each residue.
  • Col: Indicates the color applied to each residue.

Step 10: Making Selections with the Control Panel

You can use the Control Panel to make selections in your structure:

  • Click on a chain identifier to select all residues in that chain.
  • Click on a secondary structure identifier to select all residues in that piece of secondary structure.
  • Click on a residue name to select that specific residue.

Selected residues are highlighted in red in the Control Panel.

Step 11: Coloring Schemes

Deep View offers various coloring schemes to help visualize different aspects of your molecule. Access these under the "Color" menu in the main toolbar.

Some useful coloring schemes include:

  • CPK: Colors atoms by element (Carbon: White, Oxygen: Red, Nitrogen: Blue, Sulphur: Yellow)
  • Type: Colors by residue properties (Acidic: Red, Basic: Blue, Polar: Yellow, Hydrophobic: Grey)
  • Secondary Structure: Colors by structural elements (Helices: Red, Sheets: Yellow, Loops: Grey)
  • Chain: Assigns different colors to each chain in a multi-chain structure

Step 12: Applying Colors in the Control Panel

You can also apply colors using the Control Panel:

  2. Left-click in a color box to change the color for one residue.

Experiment with different coloring schemes to highlight various aspects of your molecule's structure and properties.
